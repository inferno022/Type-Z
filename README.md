# Type Z - Professional Android Keyboard

A modern, privacy-focused Android keyboard featuring advanced swipe gestures and custom font styling for professional users.

## Features

### Advanced Swipe Gesture System
- **Space Bar Swipe**: Press and hold, then swipe left/right to move cursor with precision
- **Delete Key Swipe**: Press and hold, then swipe left to delete words or entire lines efficiently
- **Smart Detection**: Automatically detects swipe direction and intent for seamless operation

### Professional Font System
- **18 Unique Font Styles**: Retro, Superhero, Scratch, Calligraphy, Comic Sans, and more
- **Professional Rendering**: Uses custom fonts with proper fallbacks for consistent display
- **Error Handling**: Graceful fallback to system fonts if custom fonts fail to load

### Technical Enhancements
- **Updated Build System**: Fixed Gradle deprecation warnings for modern development
- **Java 17 Compatibility**: Updated from Java 11 to Java 17 for latest Android SDK support
- **Enhanced Font Loading**: Robust font loading with comprehensive error handling
- **Swipe Gesture Handler**: New gesture detection system for improved responsiveness

## Installation

1. Download the latest APK from [Releases](https://github.com/inferno022/Type-Z/releases)
2. Enable "Install from unknown sources" in your phone's settings
3. Install the Type Z APK
4. Go to Settings → Language & Input → Keyboard
5. Select "Type Z Keyboard" and set it as default

## Usage

### Swipe Gestures
- **Cursor Movement**: Hold space bar, swipe left/right for precise cursor control
- **Word Deletion**: Hold delete key, swipe left for efficient word removal
- **Line Deletion**: Hold delete key, swipe left further for complete line deletion

### Font Styling
1. Open Type Z keyboard settings
2. Navigate to "Font Styles"
3. Choose from 18 different professional font styles
4. Apply and enjoy your custom text styling
