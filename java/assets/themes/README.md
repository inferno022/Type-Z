# Type Z Keyboard Themes

## Professional Themes Available

This folder contains professional color themes for Type Z Keyboard:

### Default Themes:
- **Type Z Default** - Clean blue theme
- **Type Z Pro Blue** - Professional blue
- **Type Z Pro Green** - Professional green
- **Type Z Pro Red** - Professional red
- **Type Z Pro Purple** - Professional purple
- **Type Z Pro Orange** - Professional orange
- **Type Z Pro Teal** - Professional teal
- **Type Z Pro Pink** - Professional pink
- **Type Z Pro Gray** - Professional gray
- **Type Z Pro Indigo** - Professional indigo
- **Type Z Pro Yellow** - Professional yellow
- **Type Z Pro Cyan** - Professional cyan

## Usage
Themes are automatically loaded by Type Z Keyboard. Each theme includes:
- Main color (primary)
- Accent color (secondary)
- Background color
- Text color
- Secondary text color

## Adding Custom Themes
Add your own themes by creating JSON files in this folder following the same structure.
